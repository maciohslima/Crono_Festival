# ⏱️ Cronômetro Festival de Quadrilhas

> **Status do Projeto:** 🚀 Concluído / Em Manutenção
> **Versão:** 1.1.0

Uma aplicação web moderna (PWA) desenvolvida especificamente para gerenciar, cronometrar e controlar de forma rigorosa o tempo das etapas de apresentação em um **Festival de Quadrilhas Juninas**.

---

## 📌 Sobre o Projeto

Em festivais de quadrilhas juninas, o cumprimento do tempo é um dos critérios mais rígidos de avaliação. Este projeto nasceu para fornecer uma ferramenta visual gigante, clara e de fácil operação para mesários, jurados e o público acompanharem cada fase da apresentação sem distrações.

Como o sistema é um **Progressive Web App (PWA)**, ele pode ser instalado diretamente no computador ou celular e funciona perfeitamente **offline** (sem internet) após o primeiro acesso.

---

## ✨ Funcionalidades Principais

*   **⏱️ Quatro Cronômetros Independentes:** Configuração e contagem dedicada para as fases cruciais:
    *   *Produção* (Montagem do cenário)
    *   *Apresentação* (O espetáculo principal)
    *   *Saída* (Desmontagem e retirada de materiais)
    *   *Limpeza* (Organização da quadra para o próximo grupo)
*   **⚠️ Alerta Visual de Estouro:** Quando o tempo configurado chega a zero, o painel exibe um alerta piscante de `TEMPO EXCEDIDO`, continuando a contagem em modo progressivo para contabilizar os segundos de atraso.
*   **🖥️ Interface Inteligente (Hover Trigger):** O painel de controle e botões no rodapé se escondem automaticamente para dar foco total ao cronômetro gigante, reaparecendo instantaneamente quando o usuário passa o mouse na parte inferior da tela.
*   **🖼️ Fundo Customizável:** Permite realizar o upload de uma imagem de fundo (como o logo do festival ou do patrocinador) direto pela interface.
*   **📱 Suporte a PWA e Tela Cheia:** Botão dedicado para expandir o app em tela cheia e suporte para instalação nativa no Android/iOS/Windows.
